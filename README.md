# React-icons-treeshaking
React icons library in a tree shaking manner

## To run test app 
```bash
yarn install
yarn build-lib
yarn build-run
```

## To run webpack analyzer 
```bash
yarn install
yarn build-lib
yarn build-webpack
```

## To run story Book
```bash
yarn install
yarn storybook
```

## To publish the given library
```bash
yarn install
yarn build-lib
npm publish
```
